ShadowFeed — Automação de Conteúdo para Instagram
 SaaS    IA Generativa    Pipeline Multiagente    Beta 
Sistema avançado de automação de posts carrossel para Instagram. A IA realiza pesquisa de conteúdo, cria a narrativa, gera as imagens com identidade visual e entrega os slides prontos para publicação — 100% alinhado com o nicho e a identidade do usuário.

Beta

DETALHE TÉCNICO
Pipeline multiagente com responsabilidades separadas: Agente 1 pesquisa e valida o conteúdo. Agente 2 cria o roteiro do carrossel. Agente 3 gera as imagens com coerência visual entre slides. Um orquestrador central garante que o output final seja publicável sem intervenção humana.


Stack: Python  ·  LLM Pipeline  ·  Image Generation AI  ·  Claude  ·  Gemini

GitHub: https://github.com/Kmzf777/shadowfeed-beta

JobAI — Agente de Vendas Ultra-Humanizado para WhatsApp
 WhatsApp    Vendas    Prompt Engineering Avançado 
Agente de WhatsApp para venda de produto digital. Arquitetado com prompt engineering avançado para criar uma experiência de atendimento que o usuário não percebe ser automatizada. O funil de vendas é embutido na conversa de forma completamente natural.

DETALHE TÉCNICO
A humanização opera em 4 camadas: (1) variação algorítmica do tempo de resposta conforme o contexto; (2) linguagem adaptativa que espelha o perfil do lead; (3) uso estratégico de marcadores de oralidade e pausas; (4) memória de curto prazo que garante coerência narrativa ao longo de toda a conversa — eliminando qualquer padrão mecânico.


Stack: WhatsApp API  ·  LLM  ·  Prompt Engineering  ·  n8n

GitHub: https://github.com/Kmzf777/jobai
